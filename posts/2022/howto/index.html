Theme usage

I’m goint to show you here how to configure my hugo theme. You can find the theme here. To use it, you need to make some basics configurations. Let’s go.

Config file

We’re going to configure the config file first. This file is at on the root directory of your site and named config.toml. Copy the configuration file below and make change based on the comment and you desires.

baseURL = "https://testmara.github.io/"  # change it to your base url
languageCode = "en-us"  # language to use
title = "@dilag_Luc"  # title (appear on tab of browser)
#logo = "./static/images/blog/background.png"
theme = "blogtheme"  # theme name 
googleAnalytics = "My GA code"  # your google analytics code
copyright = "dilag_Luc 2022"    # your copyright text (appear on foot)
[markup.goldmark.renderer]      
unsafe= true

rssLimit = 20                      #rss
# add your name and email if you want
[author]
	name = "dilag_Luc"
    email = ""

# modify it as you can
[params]
	description = "Description de mon nouveau super site"
	author = "dilag"
	authorImage = "/images/author.jpg"  
  twitter = "https://twitter.com/dilag_luc"
  discord = "https://discord.com/"  
  github = "https://github.com/dilagluc/dilagluc"

# it let you generate http://localhost:1313/blog/howto/ as link instead of http://localhost:1313/blog/22/09/howto/ so don't change it
[permalinks]
    blog = "/blog/:filename/"


# Nothing to change here

[related]
    threshold = 0 
    includeNewer = true
    toLower = false

    [[related.indices]]
        name = "categories"
        weight = 100

    [[related.indices]]
        name = "date"
        weight = 10

# Add new page to your blog, here for example you have 3 pages categories, About and Blog , you can add many as you want by following this example (pre is a small icon you want to appear on menu for your page)
[[menu.main]]
    name = "Categories"
    url = "/categories"
    pre = "<i class='fa fa-list-alt'></i>"

[[menu.main]]
	name = "About"
	url = "/about"
    pre = "<i class='fa fa-user-secret'></i>"
  

[[menu.main]]
	name = "Blog"
	url = "/blog"
    pre = "<i class='fas fa-blog'></i>"


# Add socials medias to your page, you can add many as you want by following this example, also here pre is a icon of social media(you need to find it via font-awesome and url is your social medial link)
[[menu.social]]
    name = "GitHub"
    url = "https://github.com/mygithub"
    pre = "<i class='fab fa-github'></i>"

[[menu.social]]
    name = "YouTube"
    url = "https://www.youtube.com/"
    pre = "<i class='fab fa-youtube'></i>"
    

[[menu.social]]
    name = "Twitter"
    url = "https://twitter.com/"
    pre = "<i class='fab fa-twitter'></i>"
    
[[menu.social]]
    name = "Discord"
    url = "https://discord.com/"
    pre = "<i class='fab fa-discord'></i>"


# here add an item you want to have on page footer, follow the examples
[[menu.footer]]
  name = "About"
  url = "/about"

[[menu.footer]]
  name = "Privacy"
  url = "/privacy"

# Nothing to  change here
lastmod = ["lastmod", ":fileModTime", ":default"]
enableGitInfo = true

Others config

Add an _index.md file in your post directory and write your text and add html if you want , this is your home page. Also add a about.md.

Write a post

Now you can start to write a new post. Navigate to your root directory and:

┌─[user@parrot]─[~/Desktop/test/mysite]
└──╼ $hugo new blog/2022/howto.md
/home/user/Desktop/test/mysite/content/blog/2022/howto.md created

Now you can open it with your editor and start editing. For the header of your makdown file:

title: "Lorem Ipsum"     # here the title of page
date: 2022-06-28T01:48:27+01:00  # don't change it
draft: true                      # change it to false if you finish to write
description: "Petite description"  # if you want
images: ["/images/blog/2022/06/jeffrey.jpg"]   # if you want but not used ( used on 1st theme version)
article: 1          # to specify that it is an article
herotext: je t'aime  # the subtitle( i dont know why i call it herotext haha)
categories:    # to specify a categories of your post
  - rbb
  - jui
  - vgygj
  - ghjg

Don’t forget the _index.md file for the home page. Now you can start writing
